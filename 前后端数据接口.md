## 共享会议室前后端数据接口
### 登录
- param
```
type:'post'
url:'/login'
data:{
    'username':'leida',
    'password':'123456'
}
```
- return
```
#成功登录
{
    'status':'success',
    'data':{
        'user_id':'1',
        'user_name':'leida',
        'user_role':'admin' #或者user(申请人)
    },
    'msg':''
}

#登录失败
{
    'status':'failure',
    'data':null,
    'msg':'username or password is wrong'#失败原因
}
```

- param

```
type:'get'
url:'/roomInfo/page/1'#传入页码
```
- return

```

    'status':'success',
    'data':[

        'room_id':'1',
        'room_name':'3-233',
        'room_desc':'多媒体',

        'room_id':'2',
```
